# Word-Lerning

AI-powered vocabulary memorization through authentic article generation.

---

## Choose Your Version

| Version | For | Language |
|---------|-----|----------|
| **[zh/](zh/)** | 学习英文的中国用户 | 中文界面 + CET-6/考研难度 |
| **[en/](en/)** | Anyone learning any language | English interface + configurable target language |

Each version is a **complete standalone package**. Choose one, download it, run `install.bat` (or `bash install.sh`), and start.

---

## What It Does

1. **Triage** your word list into familiar / unfamiliar via a web card tool
2. **Extract** target words with priority weighting (less-used words first)
3. **Search** authentic articles from quality sources
4. **Rewrite** articles to naturally embed your target words (uncommon meanings preferred)
5. **Learn** through translation exercises, vocabulary tables, and reading comprehension

All driven by Claude Code skills — just type `/init-wordlist` and `/generate-article`.

---

## Quick Install

```bash
# Chinese version
cd zh && install.bat       # Windows
cd zh && bash install.sh   # macOS / Linux

# English version
cd en && install.bat       # Windows
cd en && bash install.sh   # macOS / Linux
```
